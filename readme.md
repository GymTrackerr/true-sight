# TrueSight
Created by Daniel Kravec on January 27, 2026



### Yolo Model 
The model used for pose estimation is `yolo11m-pose.pt` from Ultralytics YOLO. This model is capable of detecting 17 keypoints on the human body.
---
- Nose
- Left Eye
- Right Eye
- Left Ear
- Right Ear
- Left Shoulder
- Right Shoulder
- Left Elbow
- Right Elbow
- Left Wrist
- Right Wrist
- Left Hip
- Right Hip
- Left Knee
- Right Knee
- Left Ankle
- Right Ankle
